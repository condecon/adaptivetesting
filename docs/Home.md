## Features

* **IRT Models**
  * 4PL, 3PL, 2PL, Rasch
  * GRM
  * GPCM
* **Ability Estimators**
  * Maximum Likelihood
  * Bayes Modal 
  * Expected a Posteriori  
* **Item Selection**
  * MFI
* **Exposure Control**
  * Randomesque
  * Maximum Priority Index
* **Content Balancing**
  * Maximum Priority Index
  * Weighted Penalty Model
  * **Stopping Criterion**
  * Standard Error
  * Test Length


## Package Installation
### PyPi

```bash
pip install adaptivetesting
```

### Conda
```bash
conda install conda-forge::adaptivetesting
```
